¿Qué es la API Key?
Esta herramienta usa inteligencia artificial de Google (Gemini) para generar contenido pedagógico. Necesitás tu propia clave de acceso, que es gratuita.

Cómo obtenerla:
1. Entrá a aistudio.google.com
2. Iniciá sesión con tu cuenta Google
3. Hacé clic en "Get API Key" → "Create API Key"
4. Copiá la clave y pegala abajo

Tu clave se guarda solo en este dispositivo. Nunca se envía a nuestros servidores.
Pegá tu clave de Google AI Studio

⚠️ Ayuda y diagnóstico de errores

1.No carga el generador: Verificá tu conexión a internet.
2.Error de API Key: Verificá la clave en aistudio.google.com. Sin espacios extra.
3.Límite de generaciones: Se renueva automáticamente a las 00:00 hs.
4.Error de login: Verificá email y clave BRB. Si persiste, contactá al administrador.
5.Sesión bloqueada en otro dispositivo: Solo 1 dispositivo por licencia. Escribí a bbbarrera1986@gmail.com para reseteo.
6.El adjunto no se procesa: Solo PDF o Word. Máximo 8 páginas o 3000 palabras.
7.El chat no responde: Verificá que tu API Key sea válida y tenga crédito disponible.
8.No guarda el historial: El navegador puede estar bloqueando localStorage (modo incógnito).
9.YouTube no abre: Verificá que tu navegador no bloquee ventanas emergentes.
10.Error al exportar .docx: Usá la opción .txt como alternativa.
11.Perfil no guarda las escuelas: Máximo 6 instituciones permitidas.
