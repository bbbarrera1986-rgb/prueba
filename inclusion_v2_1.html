Diagnóstico de Problemas
🔑 API Key inválida
SÍNTOMAError 401 o 403 al generar.
SOLUCIÓNGemini: key empieza con AIza. Claude: sk-ant-. GPT: sk-. Sin espacios. Validá desde el menú 🔑 IA.
📊 Cuota de API agotada
SÍNTOMAError 429 — "quota exceeded" o "rate limit".
SOLUCIÓNGemini: esperá 1 minuto o usá otro motor. OpenAI/Claude: recargá crédito en la consola. Cambiá de proveedor desde el selector de IA.
⏰ Licencia vencida o dispositivo no autorizado
SÍNTOMAEl login rechaza con "licencia inactiva" o "otro dispositivo".
SOLUCIÓNContactá a BRB — bbbarrera1986@gmail.com · +54 297 517-6351. Si cambiaste de equipo, pedí reset de dispositivo.
💾 Límite de 10 generaciones/día alcanzado
SÍNTOMAEl botón "Generar" no responde y el contador muestra 10/10.
SOLUCIÓNSe reinicia automáticamente a medianoche. El historial y legajos guardados no se pierden.
📁 No veo mis legajos en otro dispositivo o navegador
CAUSALos legajos se guardan en este dispositivo/navegador (localStorage). Al cambiar de equipo o navegador están vacíos.
SOLUCIÓNUsá "Exportar JSON" en el panel lateral para hacer backup, e "Importar JSON" en el nuevo dispositivo. Hacé backup frecuente.
🌐 Error de conexión / Sin red
SOLUCIÓNVerificá conexión a internet. Desactivá VPN o bloqueadores de anuncios temporalmente. Las API de IA requieren conexión activa.
📄 El documento adjunto no procesa
CAUSAPDFs escaneados (solo imagen) no tienen texto extraíble.
SOLUCIÓNUsá PDF con texto seleccionable o Word (.docx). Si el PDF es escaneado, transcribí el contenido relevante en el campo de perfil funcional.
💻 Pantalla en blanco al cargar
SOLUCIÓNCtrl+Shift+R (forzar recarga sin caché). Si persiste: F12 → Console → copiá el error y envialo a bbbarrera1986@gmail.com.
🔗 El código DUA no funciona en el Generador de Clases
CAUSAEl Generador de Clases BRB v3.5 o superior debe tener el campo "Código DUA" habilitado.
SOLUCIÓNPegá el código completo (empieza con BRBDUA:) en el campo correspondiente. No modifiques el código.
